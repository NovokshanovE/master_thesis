\begin{itemize}\setlength{\itemsep}{0.15em}
  \item Запись хранится по ключу ``адрес сети + длина префикса''; по этому

  выполняет \texttt{Upsert}/\texttt{Delete} только в выбранном шарде.
  \item Поиск по IP: перебираются возможные длины маски, строится
  кандидат-префикс и выполняется точный поиск в соответствующем ART.
\end{itemize}

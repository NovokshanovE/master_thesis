\begin{frame}[shrink=5]\frametitle{Цель и задачи}
\textbf{Цель:} разработать микросервис, поддерживающий актуальную таблицу соответствия IP-префиксов автономным системам на основе данных протокола BGP в режиме реального времени.

\vspace{0.7em}
\textbf{Задачи:}
\begin{itemize}\setlength{\itemsep}{0.25em}
  \item Спроектировать архитектуру микросервиса и
  реализовать gRPC API для потребителей и
  взаимодействие с узлом GoBGP.
  \item Разработать хранилище на основе
  шардированного ART с поддержкой полного
  снимка и потока обновлений.
  \item Обеспечить корректное построение снимка
  таблицы без пересечений сетевых префиксов.
  \item Сравнить новое решение со старым по задержке и
  пропускной способности.
\end{itemize}
\end{frame}
